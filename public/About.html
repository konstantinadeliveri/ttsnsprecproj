Text To Speech & Speech Recognition

About Us

This web application was created as a project for the course Speech and Audio Processing for Ionian University. The website is based on Speech Recognition and Speech Synthesis. The users can upload a pdf file and an audio assistant(text-to-speech) will read it to them and they can also write a text file simply by using their voice.
The code will be displayed openly in a github repository.
